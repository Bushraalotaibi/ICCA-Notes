# Cloud Foundations (ICCA)

This section covers the core cloud fundamentals you need for the ICCA exam and for real-world cloud understanding.

## Topics Covered
- Cloud fundamentals: what cloud is and why it matters
- Key characteristics of cloud computing (on-demand, elasticity, measured service, etc.)
- Shared Responsibility Model
- Service models: **IaaS vs PaaS vs SaaS**
- Deployment models: **Public vs Private vs Hybrid**
- Cloud economics (high-level): CapEx vs OpEx, pay-as-you-go
- Basic terminology used across cloud providers

## High-Yield Notes (Exam Focus)
- Be able to explain **differences** (not just definitions):
  - IaaS vs PaaS vs SaaS (who manages what?)
  - Public vs Private vs Hybrid (use cases)
  - Elasticity vs scalability (common confusion)
- Understand the Shared Responsibility Model with examples

## Quick Examples
- **IaaS:** you manage OS + apps; provider manages hardware
- **PaaS:** provider manages runtime/platform; you deploy code
- **SaaS:** provider manages everything; you just use the app
